# Tea LP（紅茶の飲み比べセット ランディングページ）

## 概要

紅茶の飲み比べセットの販売促進を想定したランディングページです。
「初めてでも違いが分かる」「手軽だけど本格的」というコンセプトを軸に、
紅茶に詳しくない人にも特徴・商品内容・価格が直感的に伝わる構成を意識して制作しました。

## URL

* デモページ（GitHub Pages）
  https://arata-moonlux.github.io/tea-lp/
  
## 制作目的

* Web制作ポートフォリオ用
* HTML / CSS の基礎設計・レスポンシブ対応の練習
* LP構成（FV → 特徴 → セット内容 → 価格 → CTA）の理解

## 使用技術

* HTML5
* CSS3

  * Flexbox
  * Grid
  * レスポンシブデザイン（SP / PC）


## 開発環境

* VSCode
* GitHub

## 工夫した点

* PC版のContentsセクションはセット感が出るよう横並びにカードを配置しました
* SP表示ではカードレイアウトを2カラムにし、情報量と視認性のバランスを意識しました
