# Tea LP（紅茶飲み比べセット ランディングページ）

## 概要

紅茶の飲み比べセットを想定したランディングページです。
「初めてでも違いが分かる」「手軽だけど本格的」というコンセプトを軸に、
商品内容・特徴・価格が直感的に伝わる構成を意識して制作しました。

## URL

* デモページ（GitHub Pages）
  [https://arata-moonlux.github.io/tea-lp/](https://arata-moonlux.github.io/tea-lp/)

## 制作目的

* Web制作ポートフォリオ用
* HTML / CSS の基礎設計・レスポンシブ対応の練習
* LP構成（FV → 特徴 → セット内容 → 価格 → CTA）の理解

## 使用技術

* HTML5
* CSS3

  * Flexbox
  * Grid
  * レスポンシブデザイン（SP / PC）

※ JavaScript は使用していません。

## 工夫した点

* SP表示ではカードレイアウトを2カラムにし、
  情報量と視認性のバランスを意識しました
* 余白は各セクション単位で管理し、全体のリズム感を重視
* 画像サイズ・行間（line-height）を調整し、
  意図しない余白が出ないよう設計

## 今後の改善点

* アクセシビリティ（見出し構造・コントラスト）の改善
* JavaScriptを用いた簡単な動きの追加
